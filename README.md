# Active Syntax Trainer

## The Problem
Many English learners have solid foundation of vocabulary and grammar rules, and they often excel at reading and writing. However, they still struggle to hold full conversations. This happens because they lack the "muscle memory" and reflexes required to construct sentences on the fly. Without a consistent, low-pressure environment to practice, it is incredibly difficult to bridge the gap between textbook knowledge and real-world fluency.

## The Solution
This project leverages AI to create a safe, 24/7 practice environment focused entirely on active output. Instead of passive reading, the AI tutor guides students through constructing their own sentences, provides instant feedback, and encourages trial and error. By prioritizing steady, individualized progress, students can finally build the conversational reflexes they need to speak English confidently.

### Key Features
* **Multimodal Input:** Support for both text and speech input to train different communication reflexes.
* **Real-Time, Scaffolding Feedback:** The AI evaluates sentences and provides gentle, constructive corrections without simply handing away the answers.
* **Dynamic Level Tracking:** The engine automatically adapts to the student's current skill level, ensuring practice is always challenging but never overwhelming.
* **Active Production Focus:** Shifts the learning model from vocabulary memorization to the active creation of thoughts and sentences.

## Future Ambitions
* **Dynamic Placement Assessment:** Building an initial onboarding evaluation that automatically determines a new student's CEFR level, placing them at the exact right starting point on the grammar roadmap.
* **Native Language Translation:** Integrating localized translation APIs to provide targeted hints in the student's native language when they struggle, reducing friction while maintaining an English-first environment.
* **Real-World Roleplay:** Simulating everyday scenarios (e.g., ordering food, job interviews, casual small talk) so students can practice highly relevant, situational English.
* **Natural Conversational Flow:** Pivoting from structured grammar exercises into free-flowing, unscripted chats about daily topics to truly test conversational reflexes.
